# 幾何学ビューアー (Geometry Viewer)

Vanilla JSとHTML5 Canvasで構築された、モジュール式で軽量な3D幾何学ビジュアライザーです。このプロジェクトでは、インタラクティブな投影を通じて、様々な数理曲面、多面体、および複雑なトポロジー的オブジェクトをデモンストレーションします。

## プロジェクト構成

### ルートファイル

- `index.html`: メインのエントリーポイント。利用可能なすべての幾何学的形状の階層的なギャラリーを表示します。
- `viewer.html`: インタラクティブなビジュアライザーページ。3Dから2Dへの投影、回転、および数学的背景や数式の表示を処理します。

### CSS (`/css`)

- `base.css`: グローバルスタイル、タイポグラフィ、およびコアとなるCSS変数（テーマ設定）。
- `gallery.css`: ギャラリーのインデックスページ専用のレイアウトとスタイリング。
- `viewer.css`: ビューアーインターフェースのスタイリング（UIオーバーレイ、情報パネル、KaTeXによる数式レンダリングなど）。

### JavaScript (`/js`)

- `gallery.js`: ギャラリーのレンダリングを制御します。`shapes`モジュールからの階層データをDOMにマッピングします。
- `geometry.js`: すべての幾何学データとカテゴリー定義を統合してエクスポートするブリッジ。
- `renderer.js`: コアとなるグラフィックスエンジン。3D頂点の2Dスクリーン座標への投影と、深度ソート（ペインターのアルゴリズム）を用いたCanvasレンダリングを実行します。
- `utils.js`: 数学的なユーティリティ関数。ベクトルの正規化や汎用的な媒介変数曲面ジェネレーターを含みます。
- `viewer.js`: 個別の形状ビューアーのメインコントローラー。アニメーションループ、マウス/タッチによる回転操作、およびUI状態を管理します。

### 幾何学的形状 (`/js/shapes`)

- `index.js`: 中央レジストリ。ギャラリーで使用される階層的なカテゴリー（学問分野 -> グループ）を定義し、すべての形状定義を統合します。
- `polyhedra.js`: プラトン立体やサッカーボールなどの切頂構造を含む、離散的な立体のデータと生成器。
- `surfaces.js`: 球体、トーラス、クラインの壺などの非向き付け可能多様体を含む、連続的な数理曲面の生成器。
- `complex.js`: フラクタル（ローレンツ・アトラクタ）、生物学的モデル（DNA）、4次元投影（テッセラクト）などの高度なオブジェクトの定義。

## 主な特徴

- **カスタム投影エンジン**: 3Dから2Dへの透視投影をゼロから実装。
- **媒介変数による生成**: 数学方程式から複雑な曲面を生成するための柔軟なヘルパー関数。
- **インタラクティブなUI**: ドラッグによる回転操作とスムーズな自動回転。
- **数学的コンテキスト**: 各形状の基礎となる数式をレンダリングするためのKaTeX統合。

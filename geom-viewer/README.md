# 幾何学ビューアー (Geometry Viewer)

Vanilla JSとHTML5 Canvasで構築された、モジュール式で軽量な3D幾何学ビジュアライザーです。このプロジェクトでは、インタラクティブな投影を通じて、様々な数理曲面、多面体、および複雑なトポロジー的オブジェクトを探索できます。

## プロジェクト構成

### ルートファイル

- `index.html`: メインのエントリーポイント。利用可能な形状の階層的なギャラリーを表示します。
- `viewer.html`: インタラクティブなビジュアライザーページ。3D投影、回転操作、数学的背景の表示を処理します。

### CSS (`/css`)

- `base.css`: グローバルスタイル、タイポグラフィ、テーマ設定。
- `gallery.css`: ギャラリーのインデックスページ専用のレイアウト。
- `viewer.css`: ビューアーインターフェースのスタイリング（ボトムナビゲーション、情報パネル、KaTeX統合）。

### JavaScript (`/js`)

- `gallery.js`: ギャラリーのレンダリング。形状データをDOMにマッピングします。
- `renderer.js`: コア・グラフィックスエンジン。3D頂点の透視投影と、深度ソート（ペインターのアルゴリズム）を用いた描画を実行します。
- `utils.js`: 数学ユーティリティ。並行移動標構（Parallel Transport Frame）を用いた高度なチューブ生成や、媒介変数曲面生成器を含みます。
- `viewer.js`: メインコントローラー。アニメーションループ、回転操作、UI状態の管理。

### 幾何学的形状 (`/js/shapes`)

- `index.js`: 中央レジストリ。学問分野ごとのカテゴリー定義と、全形状の統合。
- 個別の形状ファイル（例: `tetrahedron.js`, `torus.js`, `knot.js`, `lorenz.js`, `tesseract.js` など）: 各形状の頂点・面生成ロジックが独立したモジュールとして定義されています。

## 主な特徴

- **独自投影エンジン**: 外部ライブラリを使用せず、3Dから2Dへの透視投影をスクラッチで実装。
- **高度なチューブ生成**: `tube` ユーティリティにより、滑らかな曲線（トーラス結び目など）に対してねじれのない安定したメッシュを生成。
- **洗練されたUI/UX**: 
    - 直感的なドラッグによる回転操作。
    - 画面下部のナビゲーションバーによる形状切り替え。
    - KaTeXによる美しい数式レンダリング。
- **数学的アプローチ**: 単なるグラフィックではなく、トポロジー、微分幾何学、力学系などの数学的背景を重視。

## 操作方法

- **ドラッグ**: 形状を回転させます。
- **i ボタン**: 形状の説明と数学的な数式を表示します。
- **Gallery**: トップページへ戻ります。
- **Previous / Next**: 次または前の形状へ切り替えます。

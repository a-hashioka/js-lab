# Geometry Viewer (Modernized)

**React**、**TypeScript**、**Vite**で再構築された、高度な3Dおよび4D幾何学可視化ギャラリーです。

このプロジェクトでは、古典的なプラトン立体から複雑な4D多胞体、動的な物理シミュレーションまで、38種類以上の数学的、物理学的、生物学的な形状のインタラクティブなギャラリーを提供します。

## ✨ 主な機能

- **インタラクティブな3Dレンダリング**: 深度ソートと透視投影を備えた、独自構築のCanvasベースのレンダリングエンジン。
- **4D可視化**: 4次元から3次元への投影（テッセラクト、カラビ・ヤウ多様体など）をサポート。
- **広範なギャラリー**: 以下を含む38種類以上のカテゴリ別形状：
  - **数学**: プラトン立体、フラクタル、極小曲面、位相同型な異常。
  - **物理学**: 波動（クラドニ図形）、天体物理学モデル（シュヴァルツシルト解）、カオス（ローレンツ・アトラクタ）。
  - **化学**: 結晶格子、フラーレン。
  - **生物学**: 形態形成、微視的構造。
- **高度なメッシュ生成**:
  - **ビショップ・フレーム (Bishop Frames)**: 管状構造におけるねじれ（トーション）を最小限に抑えます。
  - **パラメトリック・マッピング**: 高精度な曲面生成。
  - **細分割曲面 (Subdivision Surfaces)**: 最適化されたメッシュの細分化。
- **レスポンシブでモダンなUI**: タッチ操作のサポートと高DPIスケーリングによるスムーズなインタラクション。

## 🛠 技術スタック

- **フレームワーク**: [React 19](https://react.dev/)
- **言語**: [TypeScript](https://www.typescriptlang.org/) (厳密な型安全性)
- **バンドラー**: [Vite 8](https://vitejs.dev/)
- **スタイリング**: Vanilla CSS (カスタムプロパティとレスポンシブデザイン)
- **レンダリング**: HTML5 Canvas API (独自ソフトウェアレンダラー)

## 🚀 はじめに

### 前提条件

- [Node.js](https://nodejs.org/) (v18以降)

### インストール

```bash
# 依存関係をインストール
npm install
```

### 開発

```bash
# 開発サーバーを起動
npm run dev
```

### 本番ビルド

```bash
# 本番用にビルド
npm run build

# 本番ビルドのプレビュー
npm run preview
```

## 📐 数学的な特徴

- **4D投影**: 透視投影 $P(x,y,z,w) = \frac{d}{d+w}(x,y,z)$ を使用して、4次元の多胞体を可視化します。
- **ビショップ・フレーム (平行移動)**: チューブ形状（DNAや結び目など）は、接ベクトルに沿って座標系をねじれなしに伝播させることで生成されます。
- **画家のアルゴリズム (Painters Algorithm)**: ソフトウェアレンダラーで正しい隠面処理を確実にするため、面は平均スケール係数によって深度ソートされます。

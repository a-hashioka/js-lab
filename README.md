# JS Lab

JS Labは、JavaScript、TypeScript、そして最新のWeb技術を用いた多様な実験やプロジェクトを一つにまとめた総合ポータルサイトです。

各プロジェクトは独立したディレクトリに整理されており、独立性を保ちながら共通のプラットフォーム上で動作します。

## 🧪 現在公開中のプロジェクト

### [Geometry Viewer](./src/projects/geom-viewer/README.md)
3Dおよび4Dの幾何学形状をCanvas APIを使用して独自レンダリングするギャラリーです。38種類以上の数学的・物理的な形状をインタラクティブに探索できます。

## 🚀 開発の進め方

### 新しいプロジェクトの追加
1. `src/projects/` 配下に新しいディレクトリを作成します（例: `src/projects/my-new-app/`）。
2. プロジェクト固有のコード（コンポーネント、ロジック、スタイル、README）をそのディレクトリ内に配置します。
3. `src/App.tsx` のルーターに新しいプロジェクトを追加し、ポータル画面にリンクを表示します。

### インストールと起動
```bash
npm install
npm run dev
```

## 🛠 技術スタック
- **Vite**: 高速な開発環境とビルド。
- **React 19**: UIコンポーネントの管理。
- **TypeScript**: 型安全な開発。

## 📂 ディレクトリ構成
```text
.
├── src/
│   ├── projects/          # 各プロジェクトのディレクトリ
│   │   └── geom-viewer/   # 幾何学ビジュアライザー
│   ├── App.tsx            # ポータルルーター
│   ├── main.tsx           # エントリーポイント
│   └── App.css            # ポータルのスタイル
└── package.json           # 共通の依存関係
```
